# Role Play

Role Play é uma técnica de prompting em que você pede para a IA assumir um papel específico ao responder.

Em vez de apenas solicitar uma tarefa, você define um contexto como:

“Atue como…”
“Você é um…”
“Responda como se fosse…”

## Exemplo básico

Prompt comum:

```
Explique o que é computação em nuvem.
```

Prompt com roleplay:

```
Atue como um professor explicando computação em nuvem para iniciantes.
```

A resposta muda completamente em linguagem e didática.

### Exemplos práticos

**1. Contexto educacional**

```
Você é um professor de programação. Explique o que é uma variável para um aluno iniciante.
```

Resultado: resposta didática e simplificada.

**2. Contexto técnico**

```
Atue como um arquiteto de software. Explique as vantagens de microserviços.
```

Resultado: resposta mais técnica e estratégica.

**3. Contexto de negócios**

```
Você é um consultor de gestão. Analise os riscos de crescer rápido demais.
```

## Por que Roleplay funciona bem

Porque ajuda o modelo a:

- Ajustar linguagem ao público
- Definir nível de complexidade
- Produzir respostas mais realistas
- Evitar respostas genéricas

Resultado: resposta analítica e executiva.

**Roleplay vs Prompt comum**

| Prompt comum             | Prompt com Roleplay      |
| ------------------------ | ------------------------ |
| Resposta neutra          | Resposta contextualizada |
| Pode ser genérico        | Mais direcionado         |
| Menos controle de estilo | Maior controle           |

## Boas práticas

- Funciona melhor quando você especifica:
- Papel (“atuar como professor”, “como advogado”, etc.)
- Público (“para iniciantes”, “para executivos”)
- Objetivo (“explique”, “analise”, “crie”)

**Exemplo bem estruturado:**

```
Atue como um professor de Data Analytics explicando para gestores não técnicos o que é KPI.
```
