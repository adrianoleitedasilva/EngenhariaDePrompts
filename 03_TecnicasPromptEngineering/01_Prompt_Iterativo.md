# Prompt Iterativo

Um prompt iterativo é uma forma de interagir com uma IA em ciclos sucessivos de refinamento, em vez de tentar acertar tudo em uma única instrução.

Em termos simples:

```
Você envia um prompt → recebe uma resposta → ajusta/melhora o prompt → recebe uma nova resposta → repete até chegar no resultado desejado.
```

### Ideia central

Ao invés de pensar “preciso escrever o prompt perfeito”, você trabalha como em um processo de melhoria contínua.

Cada iteração serve para:

- Corrigir algo que não ficou bom
- Ajustar tom, estilo ou formato
- Adicionar detalhes
- Remover excessos
- Explorar variações

### Por que isso funciona bem?

Porque a IA responde melhor quando você:

- Observa o que ela fez
- Dá feedback específico
- Direciona com clareza

É parecido com orientar um designer, redator ou desenvolvedor.

## Exemplo simples (texto)

### 1ª versão do prompt:

```
Escreva um post sobre liderança.
```

**Resposta: Genérica.**

### 2ª iteração:

```
Escreva um post sobre liderança humanizada, com tom inspirador e até 120 palavras.
```

**Resposta: Melhor, mas ainda comum.**

### 3ª iteração:

```
Inclua um exemplo prático do dia a dia e finalize com uma pergunta reflexiva.
```

**Resultado bem mais alinhado.**

## O que caracteriza um prompt iterativo?

Um prompt é iterativo quando envolve:

- Ajustes progressivos
- Feedback após cada resposta
- Refinamento de contexto
- Evolução da ideia

Não é um tipo especial de prompt — é uma estratégia de uso.

## Benefícios

- Resultados mais precisos
- Menos frustração
- Mais controle criativo
- Melhor aproveitamento da IA

### Resumo rápido

Prompt iterativo = conversar com a IA como um processo, não como um comando único.
