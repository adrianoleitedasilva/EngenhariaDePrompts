# Engenharia De Prompts

Engenharia de Prompts é a prática de projetar instruções claras e estruturadas para obter respostas melhores de modelos de IA como o ChatGPT.

## O que é um Prompt?

Um prompt é simplesmente o comando ou instrução que você fornece à IA.

**Exemplos:**

❌ Vago:

```
Fale sobre liderança
```

✅ Melhor:

```
Explique os 3 estilos de liderança mais comuns e dê exemplos práticos no ambiente corporativo.
```

## Por que Engenharia de Prompts é importante?

Porque prompts bem construídos geram:

- ✔ Respostas mais precisas
- ✔ Menos ambiguidades
- ✔ Melhor controle do formato
- ✔ Resultados mais úteis

## Estrutura de um Bom Prompt

Um prompt eficaz normalmente contém:

**1️⃣ Contexto**

Dê à IA informações relevantes.

```
Sou professor preparando uma aula para iniciantes em Python…
```

**2️⃣ Tarefa**

O que exatamente você quer?

```
Crie um exercício prático sobre estruturas condicionais.
```

**3️⃣ Formato**

Como deseja a resposta?

```
Responda em formato de lista numerada.
```

**4️⃣ Restrições / Estilo**

Defina limites ou tom.

```
Use linguagem simples e exemplos do dia a dia.
```

**✅ Exemplo Completo**

```
Sou professor preparando uma aula para iniciantes em Python.
Crie 3 exercícios sobre estruturas condicionais.
Responda em lista numerada.
Use linguagem simples.
```
